Likewise the modified sound clips for Experiment 2 can be found in the folder 'Experiment_2_files'. The results of Experiment 1 is found in the file 'Experiment_1'. Like wise the result of Experiment 2 can be found in the file 'Experiment 2'. The excel files should be downloaded raw and renamed ".xlsx".
